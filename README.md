# HalloSvijete

Android aplikacija izrađena u Javi u Android Studiju.

## Funkcionalnosti

* Registracija korisnika
* Prijava korisnika (login)
* Room baza podataka
* CRUD operacije nad korisnicima

  * Dodavanje korisnika
  * Pregled korisnika
  * Uređivanje korisnika
  * Brisanje korisnika
* RecyclerView za prikaz korisnika
* Više Activity ekrana i navigacija putem Intenta
* Dohvat vremenskih podataka za hrvatske gradove (WeatherActivity)

## Korištene tehnologije

* Java
* Android Studio
* Room Database
* RecyclerView
* Retrofit
* Open-Meteo API
